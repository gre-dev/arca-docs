---
title: "Task Templates"
description: "Save reusable task description templates so your team can create well-structured tasks faster."
icon: "clipboard-list"
---

Task Templates lets you create reusable description templates for your workspace. When creating a task, type `/` in the description field to pick a template and populate the task instantly with pre-written structure.

<Note>Task Templates is always on and cannot be disabled.</Note>
