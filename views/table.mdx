title: "Table view"
description: "A spreadsheet-style layout for scanning tasks, editing inline, and making bulk updates."
icon: "table"
---

Table view displays tasks in a grid with columns for every field: status, priority, due date, assignee, and more. It's the closest thing to a spreadsheet in Arca, and it's the best layout when you need to see many tasks at once or update multiple fields quickly.

## Key features

- **Inline editing:** click any cell to update that field directly, without opening the task
- **Resizable columns:** drag column borders to adjust widths to match what you're looking at
- **Column sorting:** click any column header to sort tasks by that field
- **Filters:** narrow down which tasks are shown using the filter menu

## When to use table view

- **Triage sessions:** quickly assign priorities, due dates, or statuses across a batch of tasks
- **Status reviews:** scan assignees and progress across an entire project at a glance
- **Planning meetings:** walk your team through upcoming tasks with all key fields visible
- **Reporting:** see all tasks with their fields laid out clearly for a project summary

<Tip>
  Press **Mod+A** to select all tasks in the view and apply bulk edits without
  opening each one individually.
</Tip>
