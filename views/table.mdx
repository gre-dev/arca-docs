---

Table view shows a flat list of tasks that match the view's filters. Tasks are displayed as list items, the same format you see inside a list.

## What you can do

Every task card in table view supports the same quick interactions as a regular list:

- **Click** a task to open its full detail page
- **Right-click** to open the context menu (change status, priority, assignee, move, duplicate, rename, delete, and more)
- **Hover and press `S`** to quickly change the status
- **Hover and press `P`** to quickly change the priority
- **Hover and press `A`** to quickly change the assignee

## When to use table view

- Review tasks from multiple lists or folders in one place
- Check the status of work across a project without navigating list by list
- Triage tasks filtered by assignee, label, or priority
