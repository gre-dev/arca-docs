View options are the controls that determine which tasks appear in a view and how they're arranged. Getting these right is the difference between a view that's useful every day and one you only open once.

## Filters

Filters restrict which tasks are shown. Common filter options:

- **List:** show tasks from one specific list
- **Assignee:** show only tasks assigned to certain people
- **Status:** include or exclude specific workflow stages
- **Labels:** show only tasks tagged with a particular label
- **Due date:** tasks due before, after, or within a date range

Combine multiple filters to get precise results. For example: "tasks assigned to me with High priority due this week."

## Sorting

Sort determines the order tasks appear. Options include: name, due date, priority, and creation date, each with ascending or descending direction.

## Grouping

In board and list views, you can group tasks by status, assignee, or priority. Grouping creates visual sections, useful in standups to see work organized by owner or workflow stage.

## Saving changes

Changes to filters, sorting, and grouping are saved automatically. The next time you open the view, it looks exactly as you left it.

<Tip>
  Create separate views for separate needs (one for daily standups, one for
  weekly planning) rather than changing a single view back and forth each time.
</Tip>
