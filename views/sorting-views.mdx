title: "Sorting views"
description: "Control the order tasks appear in a view by sorting by date, priority, or other fields."
icon: "arrow-up-z-a"
---

Sorting determines the order tasks appear in a view. Choosing the right sort makes it immediately obvious which task to work on next, without scanning the whole list.

## Sort options

| Sort field                   | Best for                                             |
| ---------------------------- | ---------------------------------------------------- |
| **Due date** (soonest first) | Daily planning: see what's most urgent first         |
| **Priority** (high first)    | Triage sessions: tackle the most critical work first |
| **Created date** (newest)    | Reviewing recently added tasks                       |
| **Name** (alphabetical)      | Finding a specific task in a long list               |

## How to change sorting

<Steps>
  <Step title="Open a view">Click a view tab in the Views section.</Step>
  <Step title="Open the Sort controls">
    Click the **Sort** control in the view header.
  </Step>
  <Step title="Choose a field and direction">
    Pick a sort field and choose ascending or descending order.
  </Step>
</Steps>

<Tip>
  Sort your personal task view by due date so you always wake up knowing what to
  work on first.
</Tip>
