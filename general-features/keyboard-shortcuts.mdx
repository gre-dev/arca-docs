Keyboard shortcuts let you create tasks, search, navigate, and manage selections without taking your hands off the keyboard. Once familiar, they significantly speed up daily workflows.

## Common shortcuts

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + K** | Open global search |
| **Cmd/Ctrl + /** | Open the keyboard shortcuts help panel |
| **N** | Create a new task (when viewing a list or board) |
| **Cmd/Ctrl + .** | Toggle showing completed tasks in list view |
| **Cmd/Ctrl + A** | Select all tasks in list view |
| **Escape** | Clear selection or close dialogs |

## View all shortcuts

Press **Cmd/Ctrl + /** at any time to open the full keyboard shortcuts reference panel inside Arca.

<Note>Shortcuts are active when the app window is focused. Most shortcuts are ignored when your cursor is in a text input field, so you can type normally without triggering them.</Note>

<Tip>Start with **N** for new task and **Cmd+K** for search. These two shortcuts alone will save you significant time during a typical workday.</Tip>
