Arca is available as a desktop app for macOS, Windows, and Linux. The desktop version includes a local cache so your data is available even when you're offline.

## Download Arca

1. Visit **https://getarca.app/download**.
2. Choose your operating system.
3. Install the app and log in with your Arca account.

## System requirements

- **macOS**: 12 or later
- **Windows**: Windows 10 or later
- **Linux**: Modern distributions with support for AppImage

## Running in a browser

If you prefer not to install the desktop app, you can also access Arca in a browser via the same URL.

<Tip>The desktop app uses the same account and stores data securely in your user profile.</Tip>
