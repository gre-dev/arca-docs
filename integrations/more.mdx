This section only documents integrations that require extra setup steps or have enough complexity to benefit from a dedicated guide.

Arca includes additional integrations that are simple enough to set up without a guide, each one has a description and instructions directly on its settings page. To browse and enable them, go to **Settings → Integrations** in your Arca workspace.
