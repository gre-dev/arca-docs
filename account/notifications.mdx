Notification preferences let you control how Arca reaches you outside the app. To open these settings, go to **Settings > Notifications**.

## Desktop notifications

Toggle desktop notifications on or off. When enabled, Arca can send system-level notifications to your desktop while the app is running.

<Note>
  Desktop notifications are only available in the Arca desktop app. They are not
  available in the web version.
</Note>

## Web notifications

In the web version, you can enable browser push notifications instead. When enabled, Arca can send notifications to your browser even when the tab is not in focus.

## Unread notification reminders

When enabled, Arca sends you an email reminder if you have unread in-app notifications that you have not checked after 30 minutes.

## News and updates

Choose whether Arca can send you occasional product news and update emails. Options: **Never** or **Allow**.

## Notification types

The **Notification types** section lets you control which events trigger notifications. Disabling a type blocks it across all channels, inbox, desktop, web, and mobile.

| Event | Triggered when |
|-------|---------------|
| Task assignments | Someone assigns you to a task |
| New comments | Someone adds a comment to a task you're subscribed to |
| Mentions | Someone @mentions you in a comment |
| Reactions | Someone reacts to a task you reported |
| Status changes | A task's status is updated |
| Priority changes | A task's priority level changes |
| Due date changes | A task's due date is set or modified |
| Start date changes | A task's start date is set or modified |
| Due date reminders | A reminder sent before a task's due date arrives |
| Start date reminders | A reminder sent before a task's start date arrives |
| Overdue tasks | A task passes its due date without being completed |

Toggle any row to turn that notification type on or off. Changes take effect immediately.

## Viewing your notifications

Your in-app notifications appear in your Inbox. Open it from the sidebar to see task assignments, comments, mentions, and start/due date reminders.
