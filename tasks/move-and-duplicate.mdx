## Moving a task

Moving a task reassigns it to a different list within the same workspace. The task keeps all its properties (status, priority, assignees, dates, labels) and appears in the target list immediately.

<Steps>
  <Step title="Right-click the task">
    In any list or board view, right-click a task card to open the context menu.
  </Step>
  <Step title="Hover over Move Task">
    Hover over **Move Task** to reveal a submenu listing every list in the
    workspace except the one the task currently belongs to.
  </Step>
  <Step title="Select the destination list">
    Click the list you want to move the task to. The task disappears from the
    current view and is immediately available in the target list.
  </Step>
</Steps>

<Note>
  **Move Task** only appears in the context menu when there is at least one
  other list in the workspace. If you only have one list, or the task is not
  assigned to any list, the option is hidden.
</Note>

### Drag to move

In list view, you can also move a task by dragging it. There are two ways to drag:

**Drag between groups**: When tasks are grouped (by assignee, status, or priority), drag a task card from one group and drop it onto another. The corresponding field updates automatically. For example, dragging a task from the "Alice" group to the "Bob" group reassigns it to Bob.

**Drag to the sidebar**: Drag a task card and drop it onto any list name in the sidebar to move the task to that list instantly. The sidebar list items highlight as you hover over them to show where the task will land.

<Note>
  Drag and drop is only available in **list view**. In board view, tasks cannot be dragged to sidebar lists, but you can still use the right-click context menu to move them.
</Note>

## Duplicating a task

Duplicating creates a new task with the same core properties as the original. The copy is placed at the top of the same list and briefly highlighted so you know exactly where it landed.

<Steps>
  <Step title="Right-click the task">
    In any list or board view, right-click a task card to open the context menu.
  </Step>
  <Step title="Click Duplicate">
    Click **Duplicate**. A copy is created instantly, no confirmation required.
  </Step>
</Steps>
