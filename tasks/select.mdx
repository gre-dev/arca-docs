## How selection works

In list view, you can select tasks using the checkboxes on the left side of each task. Selected tasks can be edited or deleted together.

### Keyboard shortcuts

- `Cmd+A` (or `Ctrl+A` on Windows): select or deselect all visible tasks.
- `Escape`: clear selection.

## Bulk actions

Once tasks are selected, a toolbar appears at the bottom of the screen with options to perform bulk actions on all selected tasks at once.

<Note>Bulk actions are not available in board view. Use list view for batch updates.</Note>
