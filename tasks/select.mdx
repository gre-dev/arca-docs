## How selection works

In list view, you can select tasks using the checkboxes on the left side of each task. Selected tasks can be edited or deleted together.

### Keyboard shortcuts

- **Mod+A**: Select or deselect all visible tasks.
- **Escape**: Clear selection.

## Bulk actions

Once tasks are selected, use the toolbar at the top of the list to:

- Change status
- Change priority
- Assign or unassign people
- Delete selected tasks

<Note>Bulk actions are not available in board view. Use list view for batch updates.</Note>
